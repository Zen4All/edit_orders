===========================================================
Optional Items for Edit Orders
===========================================================
This folder contains optional items for Edit Orders. These
"optional items" are not officially supported.

If you do find an issue and have corrected the issue, do
post the updated version of the "optional item" to the
Edit Order Support Thread on the Zen Cart forums.

===========================================================
List of "Optional Items"
===========================================================
1. "onetime discount": This is an order total module which
   allows adding an onetime discount to an existing order.
